# caloriecam

An AVS Media application built with Vite, React, Convex, and AI.

## Tech Stack
- Frontend: Vite (React + TypeScript)
- Styling: Vanilla CSS / Tailwind
- Backend: Convex (Serverless)
- Authentication: Convex Auth
- Analytics: PostHog

## Quick Start
```bash
cd web
npm install
npm run dev
```
